## Installation 

Clone repository
```bash
git clone <repository>
cd <directory>
```
Install dependencies
```bash
bundle install
yarn install
```
Create new database
```bash
rails db:drop #just for safety
rails db:create
rails db:migrate
```
Start the server
```bash
rails s
```

## Stack
 - React 17.0.1
 - Rails 6.0.3.4
 - Ruby 2.6.5p114
